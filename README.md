# 🌸 Lavender - Multi-Session WhatsApp Bot

Production-ready WhatsApp bot with multi-session support built with Baileys.

## ✨ Features

- ✅ Multi-session support
- ✅ Auto-reconnect
- ✅ Plugin system
- ✅ Sticker maker
- ✅ Media downloaders
- ✅ Auto-update from GitHub
- ✅ Group management
- ✅ Owner commands

## 🚀 Quick Start

### Deploy on Panel

1. Set startup file: `index.js`
2. Set main file: `index.js`
3. Start the bot
4. Scan QR code

### Commands

- `!menu` - Show all commands
- `!ping` - Check bot latency
- `!sticker` - Create sticker from image
- `!update now` - Update bot (owner only)

## 📝 Configuration

Edit `.env` file:

```env
BOT_NAME=Lavender Bot
BOT_PREFIX=!
```
